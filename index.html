B4A Implementation Notes
📱 Navigation System
Each screen is a separate B4A Activity. The bottom nav and back buttons call StartActivity() / Finish(). Pass category data between screens via a shared globals module.
B4A: Activity + StartActivity
🌍 Seismic Ripple Rings
Animate with a Timer firing every 50ms. Use Canvas.DrawCircle with increasing radius. Decrease alpha per ring (255 → 180 → 80 → 20) to simulate the fade-out effect.
B4A: Canvas.DrawCircle + Timer
📈 Sparkline Chart
Normalize API price values to panel height. Draw connected lines with Canvas.DrawLine. Fill the area below with a DrawPath call using a semi-transparent paint color.
B4A: Canvas.DrawLine + DrawPath
📰 News Card List
Use ListView with a custom XML layout per row (colored dot Panel, headline Label, meta Label, tag Label). Populate on JobDone after NewsAPI JSON is parsed.
B4A: ListView + CustomLayout
🗺️ Crisis Map
Load Leaflet.js inside a WebView using an inline HTML string. Inject JavaScript to drop colored markers at lat/lng coordinates received from each API response.
B4A: WebView + JavaScript injection
⚙️ Settings Toggles
Use ToggleButton for each switch row. Save preference values with KeyValueStore (B4A's built-in persistent key-value storage) on each toggle change event.
B4A: ToggleButton + KeyValueStore
🃏 Category Cards Grid
Two HorizontalLinearLayouts stacked vertically, each holding two CardViews at 50% width. Each card has an icon, two Labels, and a badge Panel in a RelativeLayout.
B4A: CardView + HLLayout
📡 API + JSON Parsing
OkHttpUtils2 fires one async GET per category on tap. JobDone event parses JSON with JSONObject/JSONArray. All four categories work independently — one failure doesn't break others.
B4A: OkHttpUtils2 + JSON
🎨 Risk Gauge Orb
DrawArc on a Canvas with SweepAngle = (score/100)*360. Overlay a smaller filled circle in the card background color to create the donut ring. Center a Label over it for the number.
B4A: Canvas.DrawArc + Label
